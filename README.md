# ListAnvil Homebrew Tap

Homebrew casks for [ListAnvil](https://listanvil.com/) apps.

## Available Casks

### ListAnvil Relay
The local desktop relay worker for ListAnvil — posts marketplace listings from your own Mac and IP. Apple Silicon (arm64) only.

To install in one step (no separate `brew tap` needed):
```bash
brew install --cask listanvil/tap/listanvil-relay
```

Or, if you've already run `brew tap listanvil/tap`:
```bash
brew install --cask listanvil-relay
```
